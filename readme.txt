Mashlahah.id WordPress Showcase Theme
Tema ini adalah versi fungsional dari simulasi live preview yang dibuat untuk showcase brand guideline Mashlahah.id.

Warna: Hijau NU, Biru Tua, Abu Terang

Tipografi: Playfair Display + Lato

Tone: Islami moderat, profesional, kredibel

Fitur: Terintegrasi dengan Gemini API untuk demo pembuatan, peringkasan, dan penerjemahan artikel.
